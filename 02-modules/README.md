# 🚀 Node.js Core Modules Demo

This project demonstrates the usage of **Node.js core modules** with real-world examples in a single file (`index.js`). It is designed for learning, practice, and interview preparation.

---

## 📌 Features

- Demonstrates **15+ Node.js core modules**
- Includes **real-world examples**
- Covers both **synchronous and asynchronous operations**
- Shows **CLI interaction, server creation, and system-level info**

---

## 📂 Modules Covered

### 🔹 Process Object

- Access command-line arguments
- Environment variables
- Memory usage
- Event loop (`nextTick`)
- Process lifecycle events

---

### 🔹 File System (`fs`)

- Create and update log files
- Read files (sync, async, promises)

---

### 🔹 Path (`path`)

- File path handling
- File parsing and resolution

---

### 🔹 OS (`os`)

- System information
- Memory, CPU, user info

---

### 🔹 URL (`url`)

- Parse URLs
- Extract query parameters

---

### 🔹 Events (`events`)

- Custom event emitter
- Event-driven programming

---

### 🔹 Stream (`stream`)

- Handle data in chunks
- Readable streams

---

### 🔹 Buffer

- Work with binary data
- Encode/decode strings

---

### 🔹 Crypto (`crypto`)

- Password hashing using SHA256
- Secure data handling

---

### 🔹 Timers

- `setTimeout`
- `setInterval`
- Background jobs

---

### 🔹 DNS (`dns`)

- Resolve domain to IP address

---

### 🔹 Assert (`assert`)

- Basic testing and validation

---

### 🔹 Util (`util`)

- Formatting strings
- Object inspection
- Promisify functions

---

### 🔹 Readline (`readline`)

- CLI interaction
- User input handling

---

### 🔹 HTTP (`http`)

- Create a simple server
- Handle routes (`/`, `/about`, etc.)

---
