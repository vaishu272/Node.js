📘 Express Concepts Used

This project demonstrates core backend concepts using Express.js. These are essential for building modern web applications.

⚙️ Middleware

Middleware runs between request and response. Used for parsing data, authentication, and logging.

🌐 Routing

Routing defines how the server responds to different URLs like /about, /contact, etc.

🔗 Route Parameters

Dynamic values passed in URL. Example: /profile/:username

🔍 Query Parameters

Used for search/filtering. Example: /product?search=laptop

📩 Form Handling (POST)

Used to send data from client to server. Access data using req.body.

📁 Static Files

Serve HTML, CSS, and images using express.static().

❌ 404 Handling

Handles unknown routes and shows a custom error page.

🚀 Fetch API

Used to fetch data from external APIs (like JSONPlaceholder).

👉 This project combines frontend + backend concepts into a complete Express.js application.
